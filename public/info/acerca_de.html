toT — Reading Meter

Manual de uso

Aplicación de escritorio para estimar el tiempo de lectura.

Versión

  • Versión: 0.1.0
  • Plataforma: Desktop (Electron)

Soporte y contacto

Reporte de bugs y sugerencias: usar el tracker de Issues del repositorio.

  • Repositorio: github.com/Cibersino/tot-readingmeter
  • Issues: github.com/Cibersino/tot-readingmeter/issues

Para facilitar soporte, incluye la versión indicada arriba y una descripción reproducible del problema.

Privacidad y datos

  • Telemetría: no se recopilan métricas ni se envían datos de uso.
  • Texto del usuario: no se transmite a servicios externos.
  • Almacenamiento: la configuración y el estado se guardan localmente en el equipo del usuario.

Conectividad: la app puede consultar un archivo público de versión en GitHub para verificar si existe una versión más reciente y puede abrir la página de releases para descargarla. No se envía el contenido del texto analizado.
