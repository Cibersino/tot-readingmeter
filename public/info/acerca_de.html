Privacidad y datos

  • Telemetría: no se recopilan métricas ni se envían datos de uso.
  • Texto del usuario: el contenido que ingresas/pegas en la app no se envía a Internet ni se comparte con servidores de los desarrolladores (se procesa localmente).
  • Almacenamiento: la configuración y el estado se guardan localmente en el equipo del usuario.
  • Actualizaciones: la app solo consulta información pública de versiones y abre el navegador para descargar; no descarga ni instala actualizaciones automáticamente.

Conectividad: solo para verificar actualizaciones en GitHub y abrir releases en el navegador. No se envía tu texto; GitHub verá tu IP (tráfico HTTPS estándar).
